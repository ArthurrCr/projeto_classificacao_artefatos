**Avaliação de Técnicas de Data Augmentation Determinísticas em Modelos de Classificação de Imagens**

---

## Descrição

Este projeto tem como objetivo implementar e avaliar o impacto de diferentes técnicas de **data augmentation determinísticas** em modelos de classificação de imagens. Utilizando bibliotecas como TensorFlow, Keras e KerasCV, aplicamos transformações específicas e controladas às imagens de entrada, permitindo uma análise precisa de como cada técnica afeta o desempenho do modelo.

---

## Objetivos

- Implementar funções de data augmentation que aplicam transformações específicas e não aleatórias.
- Melhorar a arquitetura do modelo base para aumentar sua capacidade de generalização.
- Avaliar o impacto individual de cada técnica de data augmentation no desempenho do modelo.
